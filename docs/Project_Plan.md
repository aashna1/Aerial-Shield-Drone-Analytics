PROBLEM STATEMENT: 
Drone operations generate large volumes of mission data, including flight duration, battery usage, weather conditions, payload information, and mission outcomes. Raw data alone is difficult to interpret and does not support informed decision-making.

This project develops an end-to-end analytics solution that transforms drone mission data into interactive dashboards and statistical visualizations using R and Power BI. The objective is to identify operational trends, monitor mission performance, evaluate battery efficiency, analyze environmental impacts, and generate actionable business insights for better operational planning.

PROBLEM OBJECTIVES:
• Analyze drone mission performance
• Monitor battery consumption
• Evaluate mission success rates
• Study weather impact on operations
• Compare drone models
• Analyze operator performance
• Build interactive Power BI dashboards
• Perform statistical analysis using R
