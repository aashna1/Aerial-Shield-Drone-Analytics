# 🚀 Aerial Shield Drone Analytics Dashboard

An end-to-end Drone Mission Analytics project built using **R**, **Power BI**, and **Microsoft Excel** to transform raw drone mission data into meaningful operational insights through statistical analysis and interactive dashboards.

---

## 📌 Project Overview

Modern drone operations generate thousands of mission records containing flight details, battery usage, mission status, altitude, payload information, weather conditions, and threat levels.

This project demonstrates how raw operational data can be converted into business intelligence using Exploratory Data Analysis (EDA), statistical visualizations, and Power BI dashboards.

---

## 🎯 Objectives

- Analyze drone mission data
- Identify operational trends
- Monitor battery utilization
- Evaluate mission success rate
- Compare operator performance
- Study weather impact on missions
- Build interactive dashboards
- Generate business insights
